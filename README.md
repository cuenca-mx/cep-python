### Uso

```python
from datetime import date

from cep import Transferencia

tr = Transferencia.validar(
    fecha=date(2019, 4, 12),
    clave_rastreo='CUENCA1555093850',
    emisor='90646',  # STP
    receptor='40012',  # BBVA
    cuenta='012180004643051249',
    monto=8.17,
)
pdf = tr.descargar()
```
